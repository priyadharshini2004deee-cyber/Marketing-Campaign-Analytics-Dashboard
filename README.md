# Marketing Campaign Analytics Dashboard

## Project Overview
This project is an interactive dashboard built using Python and Streamlit to analyze marketing campaign performance and customer behavior.

The system helps businesses understand:
- Customer spending patterns
- Campaign response rates
- Product category performance
- Customer segmentation by age and income

## Technologies Used
- Python
- Streamlit
- Pandas
- SQLite Database
- Matplotlib

## Features
- KPI metrics (customers, response rate, web visits)
- Customer segmentation (age, income, spending)
- Campaign response analysis
- Product performance visualization
- Interactive filters
- Business insights dashboard

## How to Run
1. Install requirements:
pip install -r requirements.txt

2. Run the app:
streamlit run marketing_campaign.py

## Dataset
The dataset is stored inside the SQLite database file `marketing.db`.
